# MixTenMod  
基于 SMAPI 的 Stardew Valley 模组  

## 简介  
MixTenMod 是一个为 Stardew Valley 设计的 SMAPI 模组，旨在扩展游戏玩法并提供更多自定义选项。  

## 安装说明  
1. 确保已安装最新版本的 SMAPI。  
2. 将本模组文件夹解压至 `Stardew Valley/Mods` 目录。  
3. 运行游戏，SMAPI 会自动加载本模组。  

## 功能特性  
- **每日收入追踪**：实时显示当日收入金额在游戏界面  
- **物品价格显示**：鼠标悬停时显示物品售价  
- **快速工具切换**：通过快捷键在常用工具间循环切换  
- **手动出售系统**：支持通过专门菜单手动出售物品 
- **支持自定义配置**：可通过 config.json 调整各项功能开关与参数  
- **兼容 Stardew Valley 1.5+**  

## 配置  
首次运行后，会在 `Mods/MixTenMod` 目录生成 `config.json`，可按需修改参数。
